# HTML (HyperText Markup Language)

## How to insert comments?
<!-- comment -->

## Tag structure

### Full tag
- Tag opening
- Content
- Tag closing
- Elements
- Attributes
<!-- 
  <h1 id="title">             // tag opening and "id" is an attribute
    Website News              // content
  </h1>                       // tag closing

  obs.: the group is an element (lines 16-18)
-->

### Empty tag
<!-- 
  <img src="" alt=""/>        // no content
-->

### Tag nesting (aninhamento de tag)
- Occurs when a tag is inserted within the content of another tag.
<!-- 
  <p>
    Hi, <strong>Ana</strong>! Let's travel tomorrow?
  </p>
-->
